# Dataset Description

### Columns
Type -
X - x coordinate of particle position (cm)

Z - z coordinate of particle position (cm)

V - Particle y axis direction cosine  
W - Particle z axis direction cosine  
Energy - 
Weight - 
Cell - 
Material -
